# MailCheckr

MailCheckr is a simple yet effective extension tool designed to validate email addresses. It offers a seamless and user-friendly approach to ensuring the accuracy of email inputs.

## Features
- **Email Validation**: Checks the format and validity of email addresses.
- **User-friendly Interface**: Easy to use and understand for all types of users.
- **Error Handling**: Provides clear messages for invalid emails.

## Installation
To install MailCheckr, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Bharathkumar-2006/MailCheckr.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MailCheckr
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To use MailCheckr, execute the following command:
```bash
node server.js
